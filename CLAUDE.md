# The Epoch Institute — Site

Multi-page historical research site. All pages share:
- /shared.css — all CSS
- /shared-data.js — EMPIRES (94), EMPIRE_PROFILES, EMPIRE_INFO, TRANSLATIONS, toSlug()
- /shared-nav.js — initNav(activePage), escHtml(), jsonp(), buildPagination(), isValidEmail()

## Pages
index.html · research.html · research-series.html · about.html · contributors.html
dataset.html · explorer.html · profiles.html · community.html · 404.html
empire/*.html — 94 individual empire pages (generated by generator.html)
logos/*.png — empire emblem images

## Key variables (shared-data.js)
EPOCH_API_URL — Google Apps Script (articles, opinions, announcements, confirmed_articles)
FORMSPREE_URL — contact + newsletter form
CONTRIB_URL — separate Apps Script for contributors

## Deployment
Namecheap cPanel (Apache). Edit files here → git add → git commit → git push → cPanel Git Version Control → Deploy Head Commit.

Replace YOURUSERNAME in .cpanel.yml with your actual cPanel username (visible in cPanel top-right or File Manager path /home/USERNAME/).

## Empire pages
Run generator.html from a local server or the live site to generate empire-pages.zip.
Extract the 94 HTML files into the empire/ folder, then commit and push.
